# Set global search (option `search`)

Set global search (option `search`)

## Usage

``` r
dt2_search_global(
  options = list(),
  value,
  regex = FALSE,
  smart = TRUE,
  caseInsensitive = TRUE
)
```

## Arguments

- options:

  Options list.

- value:

  Text.

- regex, smart, caseInsensitive:

  Search flags.

## Value

Updated `options`.
