# Checkbox input per row

Checkbox input per row

## Usage

``` r
dt2_col_checkbox(
  options = list(),
  col,
  input_id_prefix = "row_chk_",
  value_col = NULL
)
```

## Arguments

- options:

  Options list.

- col:

  Target column (name or 1-based index).

- input_id_prefix:

  Prefix for element ids (e.g., "row_chk\_").

- value_col:

  Optional boolean column to define initial state.

## Value

Updated `options`.
