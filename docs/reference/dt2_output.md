# Shiny output for DT2

Place a DT2 table in a Shiny UI.

## Usage

``` r
dt2_output(outputId, width = "100%", height = "auto")
```

## Arguments

- outputId:

  Output ID (must match the
  [`render_dt2()`](https://strategicprojects.github.io/DT2/reference/render_dt2.md)
  call in server).

- width, height:

  CSS dimensions.
