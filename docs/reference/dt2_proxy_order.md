# Order the table (proxy)

Order the table (proxy)

## Usage

``` r
dt2_proxy_order(proxy, ..., columns = NULL)
```

## Arguments

- proxy:

  [`dt2_proxy()`](https://monitoramento.sepe.pe.gov.br/DT2/reference/dt2_proxy.md).

- ...:

  Vectors `c(col, "asc"/"desc")`. If you pass `columns`, names will be
  resolved to indices.

- columns:

  Optional character vector of column names to resolve names to indices.
