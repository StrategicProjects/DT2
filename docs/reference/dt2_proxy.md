# Create a proxy for a DT2 table

Create a proxy for a DT2 table

## Usage

``` r
dt2_proxy(id, session = shiny::getDefaultReactiveDomain())
```

## Arguments

- id:

  Widget id used in
  [`dt2_output()`](https://monitoramento.sepe.pe.gov.br/DT2/reference/dt2_output.md).

- session:

  Shiny session.

## Value

A `"DT2Proxy"` object.
