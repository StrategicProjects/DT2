# Column align (Bootstrap 5 classes)

Column align (Bootstrap 5 classes)

## Usage

``` r
dt2_cols_align(options = list(), cols, align = c("left", "center", "right"))
```

## Arguments

- options:

  Options list.

- cols:

  Names or 1-based indices.

- align:

  "left","center","right".

## Value

Updated `options`.
