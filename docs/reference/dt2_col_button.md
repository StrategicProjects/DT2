# Action button per row

Action button per row

## Usage

``` r
dt2_col_button(
  options = list(),
  col,
  label = "Action",
  input_id_prefix = "row_btn_"
)
```

## Arguments

- options:

  Options list.

- col:

  Target column (name or 1-based index).

- label:

  Button label.

- input_id_prefix:

  Prefix for element ids (e.g., "row_btn\_").

## Value

Updated `options`.
