# Language helper (either list or JSON url)

Language helper (either list or JSON url)

## Usage

``` r
dt2_language(options = list(), lang_list = NULL, lang_url = NULL)
```

## Arguments

- options:

  Options list.

- lang_list:

  Named list of language keys.

- lang_url:

  URL to a JSON translation file.

## Value

Updated `options`.
