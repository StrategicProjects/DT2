# Select rows (proxy; Select extension)

Select rows (proxy; Select extension)

## Usage

``` r
dt2_select_rows(proxy, indexes, reset = TRUE)
```

## Arguments

- proxy:

  [`dt2_proxy()`](https://strategicprojects.github.io/DT2/reference/dt2_proxy.md).

- indexes:

  1-based row indices.

- reset:

  If TRUE, clear selection before selecting.
