# Simple HTML template per column (replace `{{VAL}}`)

Simple HTML template per column (replace `{{VAL}}`)

## Usage

``` r
dt2_col_template(options = list(), col, template)
```

## Arguments

- options:

  Options list.

- col:

  Name or index of target column.

- template:

  HTML string with `{{VAL}}` placeholder.

## Value

Updated `options`.
